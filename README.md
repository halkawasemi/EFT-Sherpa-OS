## 🔰 概要
**EFT Sherpa OS**は、Escape from Tarkov (PvP / PvE 両対応) プレイヤーのために開発された、軽量かつ多機能なWebベースの統合管理ツールです。

「タスクの内容確認」「フリーマーケットの価格調査」「ハイドアウトの必要素材管理」「ガンスミス構築」「レイド前のインジェクター準備」――これらを行うために、Wikiや複数のサイトを行き来する必要はもうありません。すべての情報を一つの画面に集約し、あなたのレイドと生存率を強力にサポートします。

### v0.64.7 の新機能
* **Loadout Manager**: インジェクター（注射器）のプリセット管理機能を追加。レイドに必要なコストを即座に計算します。
* **Intelligence Unit**: 収集済みアイテムの管理や、インテリジェンス情報の集約機能を強化。
* **Ballistics Live**: 弾薬データを刷新し、貫通力・ダメージに基づいたより詳細な比較が可能に。
* **Loot Manager**: レイド中に持ち帰るべきアイテムの選別をサポートする機能を強化。

## 📖 オペレーションマニュアル

### 1. ダッシュボード & データ同期
起動時、画面右上の **「SYNC」** ボタンを押してください。最新の市場価格、タスク情報、弾薬データをAPIから取得します。
* **PvE/PvP切替**: ヘッダーのバッジでモードを確認できます（現在はPvE経済に最適化）。

### 2. Market (市場監視)
フリーマーケットの価格調査を行います。
* **スマート検索**: アイテム名を入力すると、価格だけでなく「タスク/ハイドアウトでの必要数（残り/合計）」を自動判定し表示します。
* **Craft Profit**: そのアイテムが製作可能な場合、利益が出るか赤字になるかをカード上に直接表示します。

### 3. Loadout (新機能)
* **Injector Presets**: よく使う注射器のセット（例: "Labs Run", "Budget Heal"）を作成・保存できます。
* **Cost Calculation**: 現在の市場価格に基づき、そのロードアウトにかかる総額をリアルタイム計算します。

### 4. Crafts (クラフト検索)
* **逆引き検索**: 検索窓にアイテム名を入れると、そのアイテムが「成果物」になるレシピだけでなく、「材料」として使われるレシピも検索できます。
* **スマートフィルター**: 「タスクで必要なアイテムが作れるレシピ」や「ハイドアウト拡張に使うアイテム」を一発で絞り込めます。

### 5. Task Database (タスク管理)
* **進行管理**: チェックボックスで完了状況を管理できます。
* **詳細情報**: タスク名をクリックすると、必要なアイテム、報酬、そして**シェルパによる攻略アドバイス**が展開されます。

### 6. Hideout Manager (ハイドアウト管理)
* 各施設の現在のレベルを管理します。ここで設定したレベル情報は、Market画面の「必要素材判定」に自動的に反映されます。

### 7. Interactive Gunsmith (ガンスミス)
* **Auto Assemble**: 目標の「エルゴノミクス」や「反動」数値を入力して `Auto` ボタンを押すと、条件を満たす最適なパーツ構成を自動で組み上げます。
