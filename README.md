## 🔰 概要
**EFT Sherpa OS**は、Escape from Tarkov (PvEモード推奨) プレイヤーのために開発された、軽量かつ多機能なWebベースの統合管理ツールです。

「タスクの内容確認」「フリーマーケットの価格調査」「ハイドアウトの必要素材管理」「ガンスミス構築」――これらを行うために、Wikiや複数のサイトを行き来する必要はもうありません。すべての情報を一つの画面に集約し、あなたのレイドと生存率を強力にサポートします。

### 本ツールの特徴
* **インストール不要**: ブラウザだけで動作する単一のHTMLファイルです。
* **PvE価格完全対応**: PvEモードの市場価格をリアルタイム（API連携）で取得。
* **シェルパ機能**: 初心者が詰まりやすいタスクには、ベテラン（シェルパ）視点の攻略アドバイスを表示。
* **インテリジェント連携**: 「このアイテムはタスクで使う？ハイドアウトで必要？」が一目でわかるスマートバッジ機能搭載。

## 📖 オペレーションマニュアル

### 1. ダッシュボード & データ同期
起動時、画面右上の **「LIVE SYNC」** ボタン（または更新マーク）を押してください。最新の市場価格、タスク情報、弾薬データをAPIから取得します。
* **Live Stats**: 現在アクティブなタスク数や、完了すべきハイドアウトの進行度を表示します。

### 2. Market & Craft (市場・クラフト)
フリーマーケットの価格と、クラフト利益計算ができます。
* **スマート検索**: アイテム名を入力すると、価格だけでなく「タスク提出品か」「ハイドアウト要求品か」を自動判定し、バッジで表示します。
* **利益計算**: クラフトタブでは、現在の素材価格に基づいた「最も利益が出るクラフト」をランキング形式で表示します。

### 3. Task Database (タスク管理)
* **進行管理**: リスト左のチェックボックスを入れると「完了」扱いになり、次回の検索から除外できます。
* **詳細情報**: タスク名をクリックすると、必要なアイテム、報酬、そして**シェルパによる攻略アドバイス**が展開されます。

### 4. Hideout Manager (ハイドアウト管理)
* 各施設の現在のレベルを `+` `-` ボタンで管理します。
* ここで設定したレベル情報は、Market画面の「必要素材判定」に自動的に反映されます。

### 5. Interactive Gunsmith (ガンスミス)
* **Auto Assemble機能**: 目標の「エルゴノミクス」や「反動」数値を入力して `Auto` ボタンを押すと、条件を満たす最適なパーツ構成を自動で組み上げます。

## ⚠️ 注意事項

* **データ保存**: すべてブラウザの「ローカルストレージ」に保存されます。キャッシュをクリアすると進行状況が消えるためご注意ください。
* **PvE専用**: 本ツールはPvEモードの経済バランスに合わせて調整されています。
* **API**: tarkov.dev APIを利用しています。
