## 🔰 概要
**EFT Sherpa OS**は、Escape from Tarkov (PvEモード推奨) プレイヤーのために開発された、軽量かつ多機能なWebベースの統合管理ツールです。

「タスクの内容確認」「フリーマーケットの価格調査」「ハイドアウトの必要素材管理」「ガンスミス構築」――これらを行うために、Wikiや複数のサイトを行き来する必要はもうありません。すべての情報を一つの画面に集約し、あなたのレイドと生存率を強力にサポートします。

### v0.56 の特徴
* **インストール不要**: ブラウザだけで動作する単一のHTMLファイルです。
* **PvE価格完全対応**: PvEモードの市場価格をリアルタイム（API連携）で取得。
* **Wishlist（欲しいものリスト）**: 監視したいアイテムを登録し、価格変動や最安値を一覧管理可能。
* **高度なクラフト検索**: 「このアイテムは何の材料になる？」という逆引き検索や、タスク納品物が作れるレシピの絞り込みに対応。
* **インテリジェント連携**: 「あと何個必要か」が一目でわかるマトリクス表示とスマートバッジを搭載。

## 📖 オペレーションマニュアル

### 1. ダッシュボード & データ同期
起動時、画面右上の **「LIVE SYNC」** ボタン（または更新マーク）を押してください。最新の市場価格、タスク情報、弾薬データをAPIから取得します。

### 2. Market (市場監視)
フリーマーケットの価格調査を行います。
* **スマート検索**: アイテム名を入力すると、価格だけでなく「タスク/ハイドアウトでの必要数（残り/合計）」を自動判定し表示します。
* **Craft Profit**: そのアイテムが製作可能な場合、利益が出るか赤字になるかをカード上に直接表示します。
* **Wishlist登録**: アイテムカードの「☆」マークを押すと、Wishlistタブに登録できます。

### 3. Crafts (クラフト検索)
* **逆引き検索**: 検索窓にアイテム名を入れると、そのアイテムが「成果物」になるレシピだけでなく、「材料」として使われるレシピも検索できます。
* **スマートフィルター**: 「タスクで必要なアイテムが作れるレシピ」や「ハイドアウト拡張に使うアイテム」を一発で絞り込めます。

### 4. Wishlist (欲しいものリスト)
Market画面で「☆」をつけたアイテムだけを一覧表示します。
* 高額な鍵や、集めている交換素材の価格監視に最適です。

### 5. Task Database (タスク管理)
* **進行管理**: チェックボックスで完了状況を管理できます。
* **詳細情報**: タスク名をクリックすると、必要なアイテム、報酬、そして**シェルパによる攻略アドバイス**が展開されます。

### 6. Hideout Manager (ハイドアウト管理)
* 各施設の現在のレベルを `+` `-` ボタンで管理します。
* ここで設定したレベル情報は、Market画面の「必要素材判定」に自動的に反映されます。

### 7. Interactive Gunsmith (ガンスミス)
* **Auto Assemble機能**: 目標の「エルゴノミクス」や「反動」数値を入力して `Auto` ボタンを押すと、条件を満たす最適なパーツ構成を自動で組み上げます。

## ⚠️ 注意事項

* **データ保存**: すべてブラウザの「ローカルストレージ」に保存されます。キャッシュをクリアすると進行状況が消えるためご注意ください。
* **PvE専用**: 本ツールはPvEモードの経済バランスに合わせて調整されています。
* **API**: tarkov.dev APIを利用しています。
