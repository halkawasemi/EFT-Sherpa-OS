📜 EFT Sherpa OS - Patch Note v0.56

本アップデートでは、物資管理と製造ライン（クラフト）の検索機能を大幅に強化しました。
欲しいアイテムの価格監視と、複雑なクラフトツリーの把握が、スムーズになります。

✨ 新機能 (New Features)

1. Wishlist（欲しいものリスト）の実装

市場監視の負担を軽減する、ブックマーク機能を搭載しました。

機能概要: Market画面でアイテムの右側にある「⭐（星マーク）」をクリックすると、そのアイテムを監視リストに登録できます。

専用ビュー: 下部メニュー（PCはサイドバー）の 「Wishlist」 タブから、登録したアイテムだけを一覧表示できます。

価格監視: リスト内では「24時間以内の最安値/最高値」と「現在の平均価格」を一目で確認可能。買い時の判断をサポートします。

2. Craft Search & Smart Filter（クラフト検索の強化）

クラフトタブ（Crafts）が、単なる一覧表示から「検索ツール」へと進化しました。

キーワード検索: レシピの「成果物」だけでなく、「材料名」でも検索可能になりました。「この火薬、何に使える？」といった逆引きが可能です。

スマートフィルター実装: クラフトの成果物が「タスク提出品」や「ハイドアウト拡張」に必要かどうかで絞り込めるようになりました。

例：「タスクで必要なアイテムを作れるレシピ」だけを即座に表示します。

🆙 機能改善 (Improvements)

📊 Market（市場画面）の強化

アイテムカードの情報量を増やし、より戦略的な判断が可能になりました。

Craft Req（クラフト要求）フィルター: そのアイテムが「他のアイテムの材料」として使われるかどうかで絞り込めるようになりました。

Requirements Matrix（必要数マトリクス）: カード右側に、タスクとハイドアウトでの「残り必要数 / 合計必要数」をミニパネルで表示。進捗が一目でわかります。

Craft Profit表示: そのアイテム自体がクラフトで作成可能な場合、カード上に「作製時の利益（Craft Profit）」を直接表示するようにしました。買うべきか、作るべきかの判断が早まります。

使用用途の追加: 詳細画面に「USED IN CRAFT（クラフト材料としての用途）」セクションを追加しました。

📱 UI/UX（操作性）

モバイルナビゲーション: メニューに「Wish（リスト）」と「Ammo（弾薬）」を追加し、アクセス性を向上させました。

レイアウト調整: 各カードのデザインを「Boxed Style」に刷新し、視認性を高めました。

🛠 システム調整 (System)

データ処理の最適化: タスクやハイドアウトの必要数計算ロジック（calculateAllRequirements）を改良し、大量のアイテムデータを処理する際の動作を軽量化しました。

## ⚠️ 注意事項

* **データ保存**: すべてブラウザの「ローカルストレージ」に保存されます。キャッシュをクリアすると進行状況が消えるためご注意ください。
* **PvE専用**: 本ツールはPvEモードの経済バランスに合わせて調整されています。
* **API**: tarkov.dev APIを利用しています。
