## 🔰 概要
**EFT Sherpa OS**は、Escape from Tarkov (PvP / PvE 両対応) プレイヤーのために開発された、軽量かつ多機能なWebベースの統合管理ツールです。

「タスク管理」「市場調査」「弾薬選定」「装備構築」――これらを行うために、Wikiや複数のサイトを行き来する必要はもうありません。すべての情報を一つの画面に集約し、あなたのレイドと生存率を強力にサポートします。

### v0.63.1 の特徴
* **PvP / PvE 完全対応**: 設定一つでゲームモードを切り替え、各モードの市場価格をリアルタイム取得。
* **Ballistics Visualized (弾道可視化)**: 貫通力とダメージの相関図（散布図）を搭載。色分け機能により「今、自分が買える最強の弾」が一目で判別可能。
* **Loadout Manager**: インジェクター（注射器）等のプリセット作成と、トレーダーレベルに基づいたリアルタイムコスト計算。
* **Smart Economy**: あなたのトレーダーレベル（Loyalty Level）を設定することで、購入不可能なアイテムをフィルタリングし、正確なコストを算出。
* **インストール不要**: ブラウザだけで動作する単一のHTMLファイルです。

## 📖 オペレーションマニュアル

### 1. Dashboard & System Config (設定と同期)
* **LIVE SYNC**: 画面右上のボタンで、最新の市場価格、タスク情報をAPIから取得します。
* **Mode Select**: **PvP** または **PvE** モードを選択します。市場価格が選択したモードに合わせて切り替わります。
* **Trader Loyalty**: 各トレーダーの現在の信頼度（LL1～LL4）を設定します。これにより、マーケットやクラフト画面で「購入可能/不可能」が正確に判定されます。

### 2. Ballistics (弾薬性能チャート)
弾薬データを視覚的に分析し、最適な弾薬を選定します。
* **Scatter Plot**: 縦軸（貫通力）× 横軸（ダメージ）の散布図により、強力な弾薬を視覚的に特定。
* **Smart Filter**: 口径（Caliber）ごとの絞り込みや、設定したトレーダーレベルで購入できない弾薬の除外が可能。
* **Color Modes**: チャートの色分けを「貫通力別」「入手経路別」「口径別」に切り替え可能。

### 3. Loadout (ロードアウト管理)
* **Injector Presets**: 「Combat」「Loot」「Marathon」など、目的別の注射器セットを作成・管理できます。
* **Cost Calculation**: 設定したトレーダーレベルに基づき、そのセットを揃えるための最安コストをリアルタイムで計算します。

### 4. Market (市場監視)
フリーマーケットの価格調査を行います。
* **Smart Search**: アイテム名を入力すると、価格だけでなく「タスク/ハイドアウトでの必要数」を表示。
* **Craft Profit**: そのアイテムが製作可能な場合、利益が出るか赤字になるかをカード上に直接表示します。
* **Wishlist登録**: アイテムカードの「☆」マークを押すと、Wishlistタブに登録できます。

### 5. Crafts (クラフト検索)
* **逆引き検索**: そのアイテムが「材料」として使われるレシピも検索可能。
* **Smart Filter**: 「タスクで必要なアイテムが作れるレシピ」を一発で絞り込めます。

### 6. Task Database (タスク管理)
* **進行管理**: チェックボックスで完了状況を管理（ローカルストレージ保存）。
* **Sherpa Advice**: 一部のタスクには、シェルパによる詳細な攻略アドバイスや推奨装備が表示されます。

### 7. Hideout & Gunsmith
* **Hideout**: 各施設のレベルを管理し、必要な素材を逆算してMarket画面に反映させます。
* **Gunsmith**: 目標ステータス（エルゴノミクス・反動）を入力すると、APIデータに基づきパーツ構成を提案します。

## ⚠️ 注意事項

* **データ保存**: すべてブラウザの「ローカルストレージ」に保存されます。キャッシュをクリアすると進行状況や設定（トレーダーレベル等）が消えるためご注意ください。
* **API**: tarkov.dev APIを利用しています。APIサーバーの状況により、データの取得に時間がかかる場合があります。
